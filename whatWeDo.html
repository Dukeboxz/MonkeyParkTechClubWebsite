For kids who are beginners at coding we start them with a program called Scratch. Using this program they will learn about key programming concepts through creating cartoons and games by dragging and conecting blocks representing code

Once kids have got used to basic concepts we move them on to working on Micro Bits. These micro computers let the kids make programs that will use the lights and buttons on the microbits. Kids can start on the drag and drop like scratch but can also then start to program in actual coding languages such as Javascript and Python

For kids who are more advanced we have different activities each month that will let them practise coding in a fun ways . One of these activites is programming the popular game minecraft using the Python coding language
